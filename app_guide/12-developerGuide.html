3 Developer Guide

This section is intended for anyone wanting to edit or update the app. It will describe the basic code structure, compilation process and other things you may need to be aware of. A basic understanding of R shiny is required for this guide. I won’t be giving an overview of the R scripts, these have been heavily commented and should be fairly readable. If you have any questions, don’t hesitate to email me!
